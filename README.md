## 📌 Project Overview

As part of the HNG 14 Data Analytics programme, I built a fully interactive revenue and profit analysis dashboard for **TNMT Retail** using Microsoft Excel. The dashboard was designed to help business stakeholders quickly understand sales performance, identify top-performing products, and make data-driven decisions.

---

## 🛠️ Tools Used

| Tool | Purpose |
|---|---|
| **Microsoft Excel** | Data cleaning, analysis, pivot tables, and dashboard |
| **Excel Slicers** | Dynamic filtering for interactive exploration |
| **Excel Charts** | Bar charts, line charts, and donut charts |
| **Pivot Tables** | Aggregation and summarisation of sales data |

---

## 🎯 Project Objectives

- Clean and prepare raw retail sales data for analysis
- Analyse revenue and profit trends across products and categories
- Identify top-performing and underperforming segments
- Build an interactive dashboard for business stakeholders
- Generate actionable insights to support decision making

---

## 📊 Dashboard Features

- **Revenue Overview** — Total revenue and profit summary KPIs
- **Product Performance** — Top and bottom performing products by revenue
- **Category Analysis** — Revenue breakdown by product category
- **Trend Analysis** — Revenue and profit trends over time
- **Interactive Slicers** — Filter by category, product, and time period
- **Profit Margin Analysis** — Identifying most and least profitable segments

---

## 🔍 Key Insights

- Identified the highest revenue-generating product categories
- Uncovered underperforming segments with improvement opportunities
- Analysed profit margins across different product lines
- Revealed seasonal trends in sales performance
- Provided data-driven recommendations for revenue growth

3. Enable editing if prompted
4. Use the **slicers** on the dashboard to filter and explore the data interactively
